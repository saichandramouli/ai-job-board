# 🚀 AI Job Board

A modern, responsive Job Board application built using **React**, **Vite**, and **Tailwind CSS**. The application provides an intuitive platform for job seekers to discover opportunities with advanced search, filtering, bookmarking, and responsive design.

🔗 **Live Demo:** https://ai-job-board-chi-seven.vercel.app

🔗 **GitHub Repository:** https://github.com/saichandramouli/ai-job-board

---

# 📌 Project Overview

The AI Job Board is a frontend web application that simulates a modern recruitment platform. It allows users to browse available jobs, search for positions, filter opportunities based on different criteria, and bookmark favorite jobs.

The project emphasizes clean UI/UX, responsive design, component reusability, and modern frontend development practices.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/saichandramouli/ai-job-board.git
```

Navigate into the project

```bash
cd ai-job-board
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

---

# 🏗️ Build Project

Generate production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

# 🔄 CI/CD Pipeline

This project uses **GitHub Actions** for Continuous Integration.

### Workflow

On every push to the `main` branch:

1. Checkout repository
2. Install Node.js
3. Install project dependencies
4. Build the Vite application
5. Verify successful build
6. Vercel automatically deploys the latest version

This ensures only successful builds are deployed.
